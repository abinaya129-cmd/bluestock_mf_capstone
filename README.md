# Mutual Fund Analytics and Investor Insights Platform

## Overview

The Mutual Fund Analytics and Investor Insights Platform is an end-to-end data analytics project developed during the Bluestock Fintech Internship Program. The project focuses on analyzing mutual fund industry data, evaluating fund performance, generating investor insights, and building interactive dashboards for decision-making.

The platform processes multiple financial datasets, performs data cleaning, exploratory data analysis, performance analytics, and advanced analytics, and visualizes results using Power BI.

---

## Project Objectives

* Collect and process mutual fund datasets
* Clean and validate financial data
* Perform Exploratory Data Analysis (EDA)
* Calculate fund performance metrics
* Build interactive Power BI dashboards
* Generate investor insights
* Develop a fund recommendation engine
